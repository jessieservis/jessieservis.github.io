Projects

My Cows: Road Trip Wrapped

For my My Cows: Roadtrip Wrapped, I worked with two other students to create a mobile-friendly web app in under 24 hours for Spartahack X! We used Next.js for the frontend and Convex for backend and database management. The app modernizes a classic road trip game, allowing players to track cows, hazards, and game events digitally instead of with paper.

Through this project, I gained hands-on experience connecting frontend and backend systems, managing real-time updates, and troubleshooting feature behavior. I also strengthened my skills in version control, responsive design with Tailwind CSS, and prioritizing development tasks to deliver a user-focused, interactive experience. I personally drew all of the art used in this project. This project highlights my ability to combine modern web technologies with thoughtful design to create engaging applications.

Eazy Peazy Meal Planner

For this project, with a team of three classmates, I developed a meal planning application using Java and JavaFX. The app allows users to select meals from predefined categories and add their own recipes, facilitating organized meal planning. I structured the project with separate text files for different meal categories, such as desserts, entrees, and sides, to maintain clarity and ease of management.

Through this project, I enhanced my skills in Java programming, particularly in file handling and user input processing. I also learned the importance of modular design and clear documentation, which contributed to the maintainability and scalability of the application. This experience reinforced my ability to create user-friendly applications that address practical needs.
